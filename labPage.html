Introduction

The toggler() script toggles the visibility of the collapsible panel when the trigger element is clicked. If the panel is initially shown, it will be hidden. If it is initially hidden, it will be shown.

Example

Example usage with all default options:

$("h3.expand").toggler();

We can override the default class of the collapsible panels, the default toggle method, and the default duration of the animation. By default, the collapsible sections with class="shown" will be initially expanded. See the script.

Questions
